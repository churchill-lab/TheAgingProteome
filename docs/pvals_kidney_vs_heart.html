Heart vs. Kidney Comparison

Petr Simecek

Last updated: 2017-02-10

Code version: bcaf52a

There are totally 3560 gene-protein pairs detected both in heart and kidney tissues.

We used p.adjust function (Benjamini & Hochberg method) to adjust ANOVA p-values for Age and Sex effects to multiple testing. Then we used Fisher exact test (as implemented in fisher.test) to examine whether age / sex dependent genes / proteins in one tissue are also more likely to be age / sex dependent in the other tissue (\(fdr<5\%\)).

We conclude that while Age/Sex effects are shared between heart and kidney on mRNA level, those effects seem to be tissue specific for proteins.

The report has been generated from R/Markdown file pvals_kidney_vs_heart.Rmd.

 

mRNA / Age Effect

Genes changing with Age in a heart have significantly higher chance to change also in a kidney and vise verse (Fisher Exact Test, odds ratio = 3.82, p-value < 0.001).

 

mRNA / Sex Effect

Genes changing with Sex in a heart have significantly higher chance to change also in a kidney and vise verse (Fisher Exact Test, odds ratio = 1.43, p-value < 0.001).

 

Protein / Age Effect

There seems to be no statistically significant relation between whether a protein is correlated to Age in heart and kidney (Fisher Exact Test, odds ratio = 0.93, p-value = 0.343).

 

Prot / Sex Effect

There seems to be no statistically significant relation between whether a protein is correlated to Sex in heart and kidney (Fisher Exact Test, odds ratio = 1.13, p-value = 0.183).
